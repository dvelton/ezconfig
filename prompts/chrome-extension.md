I want to build a Chrome extension. Here's what I want it to do:

[Describe what you want the extension to do — for example: "add a button to every GitHub page that copies the repo URL", "highlight all prices on shopping sites", "add a sidebar with my notes on any webpage"]

Here's what I need you to do:

1. Create the Chrome extension project structure — manifest.json, any scripts, popup HTML if needed
2. Explain what each file does in plain English
3. Walk me through loading it in Chrome for testing (developer mode, load unpacked)
4. Make sure it works as expected
5. If I want to share it with others, explain my options:
   - Share the folder directly (they load it unpacked)
   - Publish to the Chrome Web Store (walk me through the process and any fees)
6. If it should also work in other browsers (Edge, Firefox), tell me what changes are needed

Keep it simple. I'm not a developer — if there's a choice between elegant and straightforward, pick straightforward.
