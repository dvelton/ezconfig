# Build a Desktop App with Electron

Paste this into your AI tool when you want to turn your web app into a desktop application.

---

I have a web app in this directory and I want to package it as a desktop application that people can download and run on their Mac or PC without opening a browser. I want to use Electron.

Here's what I need you to do:

1. Look at my project and figure out what I've built
2. Set up Electron to wrap my web app as a native desktop application
3. Make it work properly — window sizing, app title, menu bar, app icon
4. Set up a build process so I can create distributable files:
   - A .dmg for Mac
   - An .exe installer for Windows
   - Optionally a Linux build
5. Walk me through building the distributable files
6. Tell me how to share the app with others (GitHub Releases, direct download, etc.)

The app should feel like a real desktop app — proper window, appears in the dock/taskbar, can be quit normally. If there are code signing or security considerations (like macOS Gatekeeper warnings), explain what they mean and how to handle them.
