# Make It a Progressive Web App (PWA)

Paste this into your AI tool when you want your website to work like a phone app.

---

I have a web app in this directory and I want to turn it into a Progressive Web App so people can "install" it on their phones and use it like a real app — with an icon on their home screen, working offline if possible, and no app store needed.

Here's what I need you to do:

1. Look at my current project and assess what's already there
2. Add a web app manifest file (manifest.json) with the right settings — app name, icons, colors, display mode
3. Create or update a service worker so the app can work offline (cache the essential files)
4. Generate app icons in the sizes needed for iOS and Android (or tell me how to create them)
5. Add the right meta tags to my HTML so iOS and Android treat it properly
6. Test that the "install" prompt works — tell me how to test on my phone
7. If there are any gotchas (like iOS limitations), let me know

The app should feel native when launched from the home screen — no browser chrome, full screen, fast loading. Walk me through testing it on both iPhone and Android.
