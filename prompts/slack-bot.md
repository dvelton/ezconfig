# Build a Slack Bot

Paste this into your AI tool when you want to build a bot for your Slack workspace.

---

I want to build a Slack bot for my workspace. Here's what I want it to do:

[Describe what you want — for example: "respond to a /standup command with a form for daily updates", "post a daily summary of open GitHub issues", "answer questions about our company handbook", "send a welcome message to new channel members"]

Here's what I need you to do:

1. Walk me through creating a Slack App at api.slack.com
2. Set up the right permissions (bot scopes) for what I need — explain each one in plain English
3. Build the bot logic — handle commands, messages, or events as needed
4. Deploy it somewhere free:
   - For simple bots: a serverless function on Vercel, Cloudflare Workers, or Val Town
   - For more complex ones: Railway or Render free tier
5. Connect the deployed bot to Slack (webhook URLs, event subscriptions, etc.)
6. Walk me through installing it in my workspace and testing it
7. Explain how other people in the workspace can use it

Keep it on free tiers. If the bot needs to remember things between conversations, set up a simple database. Walk me through every click in the Slack admin console — I'm not a developer.
