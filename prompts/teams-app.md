# Build a Teams App

Paste this into your AI tool when you want to add custom functionality to Microsoft Teams.

---

I want to build a Microsoft Teams app. Here's what I want it to do:

[Describe what you want — for example: "a tab in our team channel that shows a project dashboard", "a bot that answers questions about our internal policies", "a message extension that lets people quickly create support tickets from a chat message", "a personal app that tracks my daily tasks"]

Here's what I need you to do:

1. Figure out what type of Teams app I need:
   - **Tab** — a custom webpage embedded in Teams (for dashboards, tools, displays)
   - **Bot** — responds to messages and commands in chat
   - **Message extension** — adds actions to the message menu (right-click on a message)
   - **Personal app** — a private tool just for the individual user
2. Set up the project using Teams Toolkit (the official way to build Teams apps)
3. Build the core functionality
4. Walk me through testing it locally in Teams
5. For deployment, explain my options:
   - Upload directly to my organization (admin or self-service)
   - Publish to the Teams app store (public)
6. If it needs a backend (API, database), set that up on a free tier

Walk me through the Teams admin console steps too — I need to know what buttons to click, not just what code to write. If my organization has restrictions on custom apps, explain what permissions I might need to request.
