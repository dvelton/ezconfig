I want to build a Word Add-in — a custom panel or button that appears inside Microsoft Word and does something useful.

Here's what I want it to do:

[Describe what you want — for example: "a sidebar that formats legal documents to our firm's style guide", "a button that generates a summary of the current document", "a panel that lets me insert boilerplate clauses from a library", "a tool that checks the document against our compliance requirements"]

Here's what I need you to do:

1. Set up the add-in project structure using the Yeoman generator for Office Add-ins (or the simplest approach available)
2. Build the task pane UI — the panel that appears inside Word
3. Connect it to Word's content — reading the document, inserting text, formatting, etc.
4. Walk me through testing it locally (sideloading in Word)
5. If I want to share it with my team, explain the options:
   - Sideload it manually on each person's machine
   - Publish to the organization's SharePoint app catalog (internal distribution)
   - Publish to the Microsoft AppSource marketplace (public distribution)
6. If it needs to call an API or external service, set that up

This should work in Word on desktop (Mac and Windows) and ideally Word on the web too. Walk me through every step — I've never built an Office Add-in before.
